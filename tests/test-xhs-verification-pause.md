# 小红书验证码/风控暂停测试案例

1. OpenCLI 返回 captcha、安全验证、请完成验证或扫码验证信号时，分类为 `VerificationRequired`；普通超时和无关“验证结果”不误判。
2. 搜索或详情已打开 crawler tab 后触发验证，不执行 `browser close`；普通错误、停止和成功仍按原规则清理。
3. 任务进入 `PAUSED`，保留已有进度，日志提示人工完成验证；自动唤醒 Chrome 失败只记录 WARNING。
4. 仪表盘显示验证提示、“打开小红书登录”和“检测登录并继续”。
5. 用户完成验证后继续原任务 ID；未完成时保持 PAUSED。
6. 不自动填写或提交账号、密码、验证码，不自动绕过风控。
