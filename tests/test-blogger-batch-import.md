# 博主白名单批量导入测试案例

## 后端

1. 下载 Excel 模板，确认固定列为博主名称、小红书用户ID、主页地址、关联城市、启用。
2. 分别导入 xlsx 与 UTF-8 csv，覆盖多个城市分隔符和启用值。
3. 首次导入返回新增数；重复上传返回更新数，博主总数不增加，城市绑定按文件替换。
4. 未知城市、缺少名称、重复身份、歧义匹配、错误格式或超过 2 MiB 时返回带行号的错误，整批无写入。

## 前端与浏览器

1. 城市 tab 不显示批量导入；切换博主白名单后显示“下载模板”和“批量导入”。
2. 选择 `.xlsx` 或 `.csv` 后自动上传，上传期间显示 loading。
3. 成功 Toast 显示新增/更新数并刷新博主列表；失败 Toast 显示后端行号原因。
4. 所有按钮使用 Element Plus 组件和图标，不使用 emoji。
