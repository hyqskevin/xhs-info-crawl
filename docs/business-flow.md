## 任务状态流转

| 状态 | 说明 | 流转条件 |
|------|------|----------|
| PENDING | 任务等待执行 | 调度触发 |
| RUNNING | 任务执行中 | 开始搜索 |
| SEARCH_DONE | 搜索完成 | 所有关键词搜索结束 |
| DOWNLOADING | 下载笔记中 | 开始下载详情 |
| PROCESSING | 清洗提取中 | OCR + 字段提取 |
| DEDUPING | 去重中 | 字段提取完成 |
| COMPLETED | 完成 | 去重与周报生成完成 |
| FAILED | 失败 | 任何阶段出现不可恢复错误 |
| PAUSED | 暂停 | 登录态失效，等待人工处理 |

`PAUSED` 状态在仪表盘提供“打开小红书登录”和“检测登录并继续”。后者登录校验成功后复用原任务 ID、首次开始时间和已有进度重新入队。

## 活动日期有效窗口

- 小红书“一周内”等选项只筛选笔记发布时间。
- 活动发生时间由后端按任务首次开始日计算，保留当天至未来 60 天内的活动。
- 已结束或超过未来 60 天的活动在入库前跳过；单个活动被跳过不影响同笔记其他活动。
- 日期无法可靠确定时保存为 `NEEDS_REVIEW`，时间为空，不进入周报。

## 活动状态流转

| 状态 | 说明 | 流转条件 |
|------|------|----------|
| RAW | 原始提取 | 字段提取完成 |
| NEEDS_REVIEW | 需人工审核 | 关键字段缺失或去重边缘 |
| DUPLICATE_CANDIDATE | 去重候选 | 系统发现相似活动 |
| MERGED | 已合并 | 人工或自动合并 |
| APPROVED | 已审核 | 人工确认可用 |
| IGNORED | 已忽略 | 非活动或低质量 |
| PUBLISHED | 已发布 | 已加入周报 |
