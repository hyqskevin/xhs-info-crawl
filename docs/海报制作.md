海报制作需求：
1. 配置中心navbar增加两个子nav，一个是现有的功能，nav名称参数配置；另一个增加功能，海报模板
2. 海报模板支持海报上传，ai识别海报的排版格式，配色，字体，emojy，背景元素，文字位置，图片位置，制作成html格式的模板
3. 有一个模板列表进行管理，支持查看模板，编辑和删除模板。
4. 新增一个navbar，海报制作。制作流程：批量选择要制作海报的活动，活动来自活动列表 -》选择海报模板 -》 填充进模板需要的内容（内容，时间范围，地点，费用等）-》人工修改优化 -》选择填充进海报的展示图 -》生成海报
5. 海报生成后，用户可以在海报列表中查看和管理生成的海报。
6. 每一次海报生成流程都作为一个海报生成任务，保存选择的活动列表，模板和引用的图片地址
